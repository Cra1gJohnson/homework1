What is the difference between a merge and a rebase?
merge allows for the combining of branches and integration of changes / differences between the two branches
Rebase allows for changes from one branch to be integrated into a separate different branch. Rebase keeps the branches separate and this is a large difference and therefore the two commands to significantly different things.








What is a remote repository?

A remote repository is a when you host your current project repository on the internet or in the cloud. This allows for access to your up to date project files from any computer that has access to the internet.
